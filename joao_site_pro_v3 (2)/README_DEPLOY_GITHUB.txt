
# Publicar via GitHub + Vercel (estático)
1) No GitHub: New repository → Add file → Upload files → **arraste o conteúdo desta pasta** (todas as subpastas).
2) Na Vercel: New Project → Import Git Repository → selecione o repo.
3) Build settings:
   - Framework: **Other**
   - Build Command: **(vazio)**
   - Output Directory: **.**
4) Deploy. A página raiz `/` redireciona automaticamente para `/pt/`.
